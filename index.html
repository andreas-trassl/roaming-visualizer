Erleben Sie die Zukunft der drahtlosen Kommunikation. In dem hier gezeigten Testaufbau wird eine PROFINET-Übertragung zur Fahrstuhlkabine im Dauertest demonstriert.

Der Testaufbau besteht aus zwei Access Points, die sich auf unterschiedlichen Etagen befinden und direkt mit einer speicherprogrammierbaren Steuerung (SPS) verbunden sind. Auf dem Fahrstuhl ist ein Client-Gerät montiert, über das PROFINET-Endgeräte angebunden werden. Über Funk wird eine kontinuierliche und stabile PROFINET-Übertragung an die Matrix aus Signalsäulen auf der Fahrstuhlkabine realisiert.

Ermöglicht wird dies durch die einzigartige Roaming-Funktionalität des Systems. Dank Zero-Latency-Roaming können Clients zwischen verschiedenen Access Points wechseln, ohne dass es zu Verbindungsabbrüchen, zusätzlichen Latenzen oder Paketfehlern kommt.
